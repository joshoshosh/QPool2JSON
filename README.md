# QPool2JSON
```
usage: parsepool.py [-h] [-i INPUT] [-o OUTPUT]

Converts NCVEC Amateur Radio question pools into JSON files.

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Filename of question pool to be inported
  -o OUTPUT, --output OUTPUT
                        Directs the JSON to a name of your choice```
